MATLAB Files
------------

Matlab files associated with the toolbox are:

collabLogLikeGradients.m: Gradient of the latent points.
collabOptionsTensor.m: Return default options for COLLAB model with a tensor
rbfadditionalKernDiagGradX.m: Gradient of RBF with side information kernel's
demMovielens2.m: Try collaborative filtering on the large movielens data.
demMovielens4.m: Try collaborative filtering on the large movielens data.
collabLoadData.m: Load a collaborative filtering dataset.
rbfadditionalKernDiagCompute.m: Compute diagonal of RBF side information kernel.
rbfadditionalKernGradient.m: Gradient of RBF with side information kernel's parameters.
demNetflix5.m: Try collaborative filtering on the netflix data.
demMovielensOrdered1.m: Try collaborative filtering on the large movielens data.
demMovielens3.m: Try collaborative filtering on the large movielens data.
collabLogLikelihood.m: Compute the log likelihood of a COLLAB.
collabOptions.m: Return default options for COLLAB model.
demMovielens1.m: Try collaborative filtering on the large movielens data.
rbfadditionalKernExpandParam.m: Create kernel structure from RBF kernel's parameters.
collabDisplay.m: Displays the provided collaborative filter model.
rbfadditionalKernGradX.m: Gradient of RBF kernel with respect to input locations.
collabOptimiseOptions.m: returns default options for collaborative filter optimisation.
collabCreate.m: Create a COLLAB model with inducing varibles/pseudo-inputs.
demNetflix1.m: Try collaborative filtering on the netflix data.
demMovielensSmall1.m: Try collaborative filtering on the small movielens data.
collabToolboxes.m: Load in the relevant toolboxes for collaborative filtering.
demNetflix2.m: Try collaborative filtering on the netflix data.
collabOptimise.m: Optimise the collaborative filter.
rbfadditionalKernParamInit.m: RBF kernel with side information.
demNetflix3.m: Try collaborative filtering on the netflix data.
rbfadditionalKernCompute.m: Compute the RBF kernel given the parameters and X.
collabPosteriorMeanVar.m: Mean and variances of the posterior at points given by X.
